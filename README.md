# Alarme com Sensor de Presença

- Projeto utilizando Arduino, sensor ultrassônico e buzzer para detectar a presença e emitir alerta sonoro.

## Componentes

- Arduino Uno
- Sensor Ultrassônico
- Jumpers
- Buzzer
- Protoboard

## Funcionamento

O sensor ultrassônico é capaz de medir distâncias. Quando algo é detectado dentro dessa distância determinada, o buzzer dispara um efeito sonoro.
